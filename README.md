# trading

여러 시장 데이터/차트 실험용 레포입니다.

## 프로젝트

- `fear_greed`
  - 공포탐욕 관련 데이터/시각화 작업
- `ticker_comparison`
  - 기본값은 `QQQ / TQQQ`이며, 상단 검색으로 다른 2개 티커도 즉시 비교 가능한 Next.js 앱

## 주요 경로

```bash
cd /Users/nam/trading/ticker_comparison
```

```bash
npm install
npm run build
```

기본 장기 데이터 파일은 각 프로젝트 내부 문서를 참고해 갱신하시면 됩니다.
